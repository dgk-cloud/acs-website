# ACS Website V2

Website for Accelerated Cloud Storage company.

## Features
- Next.js application with app router
- Complete UI components library
- Blog, contact, docs, legal, and pricing pages
- Globe animation components
- Responsive design

## Development
```bash
npm install
npm run dev
```
